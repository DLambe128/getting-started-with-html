Dan's Notes: Lesson 1 Basics of the world wide, browsers and servers

The “world wide web” is a collection of HyperText Markup Language (HTML) documents. Although web pages can can contain many types of files (such as plain text, HTML, images, videos, and music), HTML is the main component of most web pages. The links between web pages are called hyperlinks.

Your browser (a program that runs on your computer, such as IE and Firefox) makes HTTP-protocol requests via to the internet (the world’s largest computer network) to servers (computers that host the files that make up the web), and then the servers respond with files (HTML, videos, images, etc.) that your then browser displays.

More on HTML

HTML consists of:
-text content (what the user sees),
-markup (what it looks like),
-references to other documents (such as images and videos),
-and links to other pages.

More on Markup

This line is just plain text. You can recognize markup in an HTML document by the tags that surround it. In HTML, the entire unit containing a tag, its associated closing tag, and the content within the tag pair is called an element.

Tags!

The bold tag pair consists of the letter b enclosed within a less than (<) and a greater than symbol (>), and it causes content to become bold like this.The em, or emphasis, tag pair italicizes text. The em tag pair consists of the letters 'em' enclosed within the usual symbols. If you don't include a closing tag for the 'em' tag, everything after the opening tag will be italicized (until you put in a closing tag).

Certain tags, like the anchor (or just 'a' tag) can contain one or more attributes. Attributes are set equal to a value enclosed in quotation marks. Anchor tags are used specifically for creating links to other files. This link back to the udacity web-site was created with an anchor tag pair with an 'href' attribute.

The 'img' tag is used to link out to an image. The image tag usually contains 'src' and and 'alt' attributes. The 'alt' attribute value is what a user will see if the image (set equal to the 'src' attribute) doesn't load. Unlike other tags, the 'img' tag does not use a closing tag! Lone tags (that don't have a closing tag) are called void tags. Below you will see an image retrieved with the 'img' tag.

Spacing

In HTML, all white space (new lines, tabs, spaces) turns into a single space. The break, or 'br' tag, can be used to force a line break (analagous to hitting the 'Enter' key on a word document). Technically speaking, the 'br' tag simply ends a line, whereas the 'p' tag makes an invisible box (with height and width) around its contents. 'p' elements also use an opening and closing tag, whereas the 'br' tag is a void tag. The contents of this paragraph are enclosed within paragraph tags!

Inline vs. Block

Inline elements: 'b', 'em', 'br', 'img', 'a', 'strong' and 'span'
Block elements: 'p', 'div' and 'form'
